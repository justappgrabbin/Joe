# Joe Morph Substrate

Joe is the Morph Ingestion Engine: a React and Supabase substrate for uploading files, ZIP archives, code, documents, and media into shareable project spaces.

## Current target

- Ingest single files and ZIP archives
- Detect file types
- Extract lightweight structure from code and documents
- Store spaces and files in Supabase
- Generate shareable space slugs
- Morph the interface based on uploaded material

## Setup

1. Run `supabase/morph-schema.sql` in Supabase.
2. Add Supabase credentials to environment variables.
3. Install dependencies.
4. Run the Vite app.

This repo is the seed. Foundry comes later.
